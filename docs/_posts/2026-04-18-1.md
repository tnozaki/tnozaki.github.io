```
Connection closed by 217.197.84.140 port 22
```

```shell
ssh -vvvT git@codeberg.org
```

という結果になる、つまりこれ単純にサーバー側でタイムアウトが凶悪に短く設定されてて、俺がクソ長いパスフレーズをモタモタ手打ちしてると間に合わねえってことかオイ！？

Linux と Windows で発生しなかったのでオレオレ N6 だけの問題かと思ってたのだが、Bitwarden SSH-Agent が常時動いてたから問題起きなかっただけで無効にしたら等しく再現するようになった。
またパスフレーズを手打ちでなくコピペで貼っつけても問題起きないので、原因はタイムアウトということらしい。
